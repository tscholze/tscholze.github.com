Surface Duo related projects

Motiviation

Foldable devices, in this case the Surface Duo, are a tiny niche. As an advocate of iPhone and iPad, I have to admit that it is exciting to try other form factors of devices. It's something new than the eternal same. What is a negative point for the business field is a real blessing for pet projects. The still very small community of developers creates much closer ties between, for example, the Microsoft developer expierence team.

Projects

#rTsd was work my very first Xamarin.Forms Android project which is intended to feature most of the new features that makes the Microsoft Surface Duo phone unique. It uses the awesome German Dr. Windows as data source for its articles.


DuoBahn for Surface Duo will be my first Kotlin Compose-based Android app for foldable devices that uses the federal German Autobahn API as a data source.

ToboT - Google Assistant-controlled car built with Android Things and Firebase

ToboT

ToboT is an Android Things application to move a vehicle based on the Pimoroni STS Pi Kit. It also contains different other remote control methods like a website that is hosted on the vehicle or simple Google Assistant voice commands to let it move it ahead, left, right, backwards or it lets the vehicle stop. Tobot was never be intend to be more than a great and enjoyable sparetime project. That means, no code is production-ready.
